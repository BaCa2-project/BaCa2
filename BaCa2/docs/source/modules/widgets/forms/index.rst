Forms
=====

Package containing all form and form widget classes, as well as any related components. Forms are classes inheriting from ``django.forms.Form`` used to generate and validate HTML forms. Form widgets are classes inheriting from ``widgets.base.Widget`` used to wrap form instances and dictate how they are rendered.
